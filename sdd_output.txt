1.  Introduction  
 
1.1.   Purpose  
 Elementopia  is  a  gamified  chemistry  learning  system  for  high  school  students.  It  enables  students  
to
 
learn
 
chemistry
 
concepts
 
by
 
combining
 
elements
 
to
 
create
 
compounds
,
 
reinforcing
 
their
 
understanding
 
of
 
chemical
 
reactions,
 
element
 
properties,
 
and
 
periodic
 
table
 
relationships
 
in
 
an
 
interactive
 
way.
  
1.2.   Scope  
Elementopia  is  an  interactive  web  application  designed  for  high  school  chemistry  students.  It  integrates  
game
 
mechanics
 
to
 
enhance
 
learning,
 
covering
 
topics
 
such
 
as
 
atomic
 
structures,
 
bonding,
 
periodic
 
table
 
interactions,
 
and
 
simulations.
   Target  Users :  High  school  students  learning  basic  chemistry.      ●
       
Core  Features :    ○
      
Elemental  Resonance  Domain  Puzzle  Rooms  —  Students  synthesize  chemical  
compounds
 
to
 
trigger
 
environmental
 
reactions
 
that
 
clear
 
physical
 
obstacles
 
blocking
 
their
 
progression.
 
   ○
     
Meaningful  Byproduct  Diagnostics  —  Replaces  binary  "Wrong"  feedback  with  contextual  
chemistry-based
 
guidance
 
for
 
incorrect
 
compound
 
combinations.
 
  
  ○
     
Hazmat  Protocol  Failsafe  —  Disables  irrelevant  elements  after  5  failed  attempts  within  15  
seconds
 
to
 
reduce
 
random
 
guessing.
  ○
      
Randomized  Compound  Selection  —  Compounds  are  drawn  from  a  predefined  pool  per  
session
 
to
 
reduce
 
repetitive
 
memorization.
 
   ○
     
Automated  Session  Performance  Logging  —  Tracks  compound  accuracy  and  completion  
speed
 
to
 
Supabase
 
within
 
under
 
2
 
seconds
 
per
 
session.
 
  
  ○
    
Visual  Mastery  Dashboard  —  Enables  students  to  independently  identify  knowledge  gaps  
without
 
teacher
 
intervention.
 
  ○
    
Opponent-Based  Challenge  Module  —  Peer-to-peer  competitive  matchmaking  via  
auto-generated
 
5-digit
 
access
 
codes
 
with
 
connection
 
time
 
under
 
5
 
seconds.
 
   ○
   
 Speed-to-Compound  Evaluation  —  Real-time  match  winner  determination  based  on  
reaction
 
path
 
efficiency
 
via
 
Supabase
 
Realtime.

2.  Architectural  Design  
 
System  Overview
 
Elementopia  follows  a  three-tier  architecture:  
Frontend:  Developed  using  React.js  and  Vite.  
Backend:  Built  with  Spring  Boot  (Java)  exposing  RESTful  APIs.  
Database :  Supabase  (PostgreSQL)  for  persistent  storage  and  real-time  synchronization.  
Technology  Stack  
●  Frontend:  React.js,  Vite,  Material  UI  ●  Backend:  Spring  Boot,  Hibernate  (JPA)  ●  Database:  Supabase  (PostgreSQL)  ●  Communication:  REST  API,  Supabase  Realtime  for  real-time  updates
